## 💡 서비스 소개
- 서비스명 : 맛나는 맛남
- 판매상품 : 맛집 탐방을 취미로 즐기는 사람들을 연결해주는 커뮤니티
- 고객 : 홀로 맛집을 가고 싶지 않으며, 사람들과 함께 가고 싶어하는 유저들. 다양한 인맥을 넓히고자 모임을 원하는 사람들. 
- 컨셉
  - 원하는 맛집을 호스트가 주최하고, 게스트가 직접 참여할 수 있도록 하며, 호스트와 게스트는 서로 만남을 가짐과 동시에 맛있는 음식을 먹으며 대화를 하며 친목을 다질 수 있다.
